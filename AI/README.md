# AI Interview Simulator 

Інтерактивний симулятор технічних співбесід з AI-генерацією питань та миттєвим фідбеком. Проект створений для демонстрації навичок роботи з AI API та створення складних React додатків.

## Що це?

AI Interview Simulator - це повноцінний веб-додаток, який допомагає розробникам практикувати технічні співбесіди. Використовує Claude API (Anthropic) для генерації релевантних питань та надання детального фідбеку на відповіді.

## Ключові можливості

### 1. Вибір типу співбесіди
- Frontend Developer (React, Vue, Angular, CSS, HTML)
- Backend Developer (Node.js, Python, Databases, APIs)
- Full Stack Developer (комбінація технологій)
- Data Scientist (ML, статистика, Python)
- DevOps Engineer (CI/CD, Docker, Kubernetes)
- ML Engineer (Deep Learning, TensorFlow, PyTorch)

### 2. Рівні складності
- **Junior** - базові питання для початківців
- **Middle** - питання середньої складності
- **Senior** - складні архітектурні та системні питання

### 3. AI-генерація питань
- Контекстуальні питання на основі обраної спеціалізації
- Адаптація до рівня складності
- Різноманітність тем (алгоритми, архітектура, best practices)

### 4. Інтелектуальний фідбек
- Детальний аналіз відповіді
- Оцінка сильних сторін
- Рекомендації щодо покращення
- Оцінка від 1 до 10

### 5. Історія співбесіди
- Збереження всіх питань та відповідей
- Можливість переглянути попередні питання
- Фінальний звіт з усіма відповідями та фідбеком

## Технічний стек

### Frontend
- **React 18** - з використанням Hooks (useState, useEffect, useRef)
- **Функціональні компоненти** - сучасний підхід
- **CSS-in-JS** - inline стилі з CSS змінними для темізації

### API Integration
- **Anthropic Claude API** - Claude Sonnet 4
- **Асинхронні запити** - fetch API
- **Обробка помилок** - try/catch блоки
- **Conversation history** - для контекстуальних питань

### State Management
- Локальний стан компонента (useState)
- Управління історією відповідей (масив об'єктів)
- Контроль етапів (setup → interview → results)
- useRef для автоскролу

### UX Features
- Responsive дизайн
- Loading стани
- Disabled стани для кнопок
- Автоматичний скрол до фідбеку
- Складані секції для історії
